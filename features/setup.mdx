---
title: "Device Setup"
description: "Configure physical device profiles for POS terminals, KDS, Queue displays, and more."
---

A **Setup** is a named configuration profile for one physical device. Each device type (POS, KDS, Queue, Go, Cliq) has its own set of configurable settings.

## Device Types

| Type | Purpose |
|------|---------|
| **POS** | Main point-of-sale terminal |
| **KDS** | Kitchen display screen |
| **Queue** | Customer-facing queue / order-ready display |
| **Go** | Mobile order-taking device |
| **Cliq** | Self-service kiosk |

## Create a Setup Profile

<Steps>
  <Step title="Go to Manage → Setup">
    Click **Setup** in the left sidebar under Manage.
  </Step>
  <Step title="Click New Setup">
    Select the **Device Type**, then give it a **Name** (e.g. `Main Counter POS`, `Bar KDS`).
  </Step>
  <Step title="Configure settings">
    Each device type has its own settings panel — receipt printer, display options, order mode defaults, etc. Fill in the relevant fields.
  </Step>
  <Step title="Save">
    Click **Create**.
  </Step>
</Steps>

## Apply to a Device

On the physical device, open the app settings and enter the **Setup Code** shown on the profile. The device will load the configuration automatically.

## Edit a Setup

Click **Edit** on any setup row. Changes apply to the device the next time it syncs or restarts.
