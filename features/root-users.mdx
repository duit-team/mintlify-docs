---
title: "Root Users"
description: "Manage platform-level administrator accounts."
---

<Warning>
  Root users have **full access** to all tenants and platform settings. Create these accounts with care.
</Warning>

**Root Users** are platform-level administrators who can manage tenants, database servers, and global configuration. They are separate from tenant users.

## Create a Root User

<Steps>
  <Step title="Go to Root Users">
    Click **Root Users** in the root admin sidebar.
  </Step>
  <Step title="Click New Root User">
    Enter a name, email, and password.
  </Step>
  <Step title="Save">
    Click **Create**. The user can immediately log in at the root admin login page.
  </Step>
</Steps>

## Edit or Deactivate

Click **Edit** to change a root user's details. Toggle **Active** off to suspend access without deleting the account.
