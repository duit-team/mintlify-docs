---
title: "Bulk Pricing"
description: "Override product prices, names, and availability for a specific context."
---

**Bulk Pricing** lets you set a different price, display name, or availability for products in a specific sales context (e.g. a happy-hour menu or a delivery platform with different prices).

## How It Works

Each row in the bulk pricing grid represents one product. You can override:

| Field | What it changes |
|-------|----------------|
| **Price** | The price charged in this context only |
| **Name** | The display name shown to the customer (e.g. "Large Meal Deal") |
| **Active** | Whether the product is available in this context |

Changes here do **not** affect the base product — only the selected context.

## Apply Overrides

<Steps>
  <Step title="Go to Manage → Bulk Pricing">
    Select the context (e.g. *Delivery*, *Happy Hour*) from the dropdown at the top.
  </Step>
  <Step title="Find the product">
    Use the search or scroll to find the product row.
  </Step>
  <Step title="Enter overrides">
    Click into the price or name cell and type the override value. Toggle the active switch to hide a product in this context.
  </Step>
  <Step title="Save">
    Click **Save** at the top of the page. All changes for this context are saved together.
  </Step>
</Steps>

<Tip>
  Leave a field blank to inherit the default value from the base product.
</Tip>
