---
title: "Order Modes"
description: "Define how orders are placed — dine-in, takeaway, delivery, and more."
---

**Order Modes** describe the type of service for a transaction (e.g. *Dine In*, *Takeaway*, *Delivery*). They appear as a selection step when opening a new ticket on the POS.

## Create an Order Mode

<Steps>
  <Step title="Go to Manage → Order Modes">
    Click **Order Modes** in the left sidebar under Manage.
  </Step>
  <Step title="Click New Order Mode">
    Fill in:
    - **Code** — e.g. `DINE_IN`, `TAKEAWAY`.
    - **Name** — display label on the POS, e.g. `Dine In`.
    - **Active** — toggle off to hide from the POS.
  </Step>
  <Step title="Save">
    Click **Create**.
  </Step>
</Steps>

## Reorder

Drag rows using the sort handle to change the order they appear in the POS selection screen. The topmost active mode is the default.
